# 0xmortuex — Personal Portfolio

My developer portfolio showcasing AI-powered tools, cybersecurity projects, and open source work.

![Screenshot](https://via.placeholder.com/1200x630/050505/00ff88?text=0xmortuex+Portfolio)

## Built With

- Vanilla HTML, CSS, JavaScript
- No frameworks, no dependencies
- Google Fonts (Space Mono, Outfit, JetBrains Mono)

## Features

- Dark, minimal editorial design
- Interactive terminal Easter egg (press `` ` `` to open)
- Scroll-triggered animations with Intersection Observer
- Responsive across all breakpoints
- Project filtering by category
- Animated stat counters
- Matrix rain effect
